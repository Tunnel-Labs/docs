## Exports

Exports from the `@tunnel/react` package.

### TunnelToolbar

Adding the `TunnelToolbar` component to your app allows you to use Tunnel's
collaboration and feedback tools from hosted environments.

#### Parameters

<ParamField path="projectId" required>
  The ID of the project you want to connect to.
</ParamField>

<ParamField path="branch">
  The git branch for the current environment.
</ParamField>

#### Example

The example below demonstrates how you might conditionally render the `TunnelToolbar` component in your staging environment.

```jsx App.jsx
import { TunnelToolbar } from "@tunnel/react";

export default function App() {
  return (
    <>
      <h1>My App</h1>
      {process.env.APP_ENV === "staging" && (
        <TunnelToolbar 
          projectId="YOUR_PROJECT_ID"
          branch="BRANCH_NAME"
        />
      )}
    </>
  );
}
```
