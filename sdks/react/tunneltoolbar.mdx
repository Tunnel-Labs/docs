---
title: "TunnelToolbar"
description: "The `TunnelToolbar` component allows you to add the Tunnel Toolbar to your hosted app."
---

## Parameters

<ParamField path="projectId" required>
  The ID of the project you want to connect to.
</ParamField>

<RequestExample>
  ```jsx App.jsx 
  import { TunnelToolbar } from '@tunnel/react';

export default function App() {
  return (
    <>
      <h1>My App</h1>
      <TunnelToolbar projectId="YOUR_PROJECT_ID" />
    </>
  );
}

```
</RequestExample>
```
