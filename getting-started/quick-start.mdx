## Find your Project ID
To copy your Project ID, navigate to the [Tunnel Dashboard](https://tunnel.dev/projects), select your project, and go to **Settings** to copy your project ID.

## Add the Tunnel Toolbar to your app
To enable Tunnel in your app, add the `<TunnelToolbar />` component to your app. For now, we have specialized SDKs for React and Next.js apps, but if you're using another framework, you can still use Tunnel by adding the `<script>` tag to your app.
<Tabs>
  <Tab title="React">
    ### Install the Tunnel React SDK
    For more information, see the [React SDK Docs](/sdks/react).
    <ReactSDKInstall />

    ### Add the `<TunnelToolbar />` component to your app

    To enable Tunnel in your app, conditionally render the `<TunnelToolbar />` component in your app. Developers usually only render the `<TunnelToolbar />` component in your app when it is running in a preview environment.

    The `<TunnelToolbar />` component requires a `projectId` prop.

    - `projectId` is the Project ID that you copied from the Tunnel Dashboard.

    ```jsx App.jsx
    import { TunnelToolbar } from '@tunnel/react';

    export default function App() {
      return (
        <div className="App">
          {(process.env.NODE_ENV !== "development") && (
            <TunnelToolbar
              projectId={process.env.TUNNEL_PROJECT_ID}
            />
          )}
        </div>
      );
    }
    ```
  </Tab>
  <Tab title="Next.js">

    ### Add the `<TunnelToolbar />` component to your app

    To enable Tunnel in your app, conditionally render the `<TunnelToolbar />` component in your app. Developers usually only render the `<TunnelToolbar />` component in your app when it is running in a preview environment.

    The `<TunnelToolbar />` component requires a `projectId` prop.

    - `projectId` is the Project ID that you copied from the Tunnel Dashboard.

    ```jsx layout.jsx
    import { TunnelToolbar } from '@tunnel/next';

    export default function RootLayout({ children }) {
      return (
        <html lang="en">
          <body>
            {children}
            {(process.env.NODE_ENV !== "development") && (
              <TunnelToolbar
                projectId={process.env.TUNNEL_PROJECT_ID}
              />
            )}
          </body>
        </html>
      );
    }
    ```
  </Tab>
  <Tab title="Other frameworks">
    ## Add the `<script>` tag to your app

    To enable Tunnel in your app, add the `<script>` tag to your app. Developers usually only add the `<script>` tag in your app when it is running in a Coherence Branch Preview environment.

    The `<script>` tag requires a `data-project-id` attribute.

    - `data-project-id` is the Project ID that you copied from the Tunnel Dashboard.

    ```html index.html
    <script
      src="https://tunnelapp.dev/__tunnel/script.js"
      data-project-id="PROJECT_ID"
    ></script>
    ```
  </Tab>
</Tabs>

## Deploy!
You're all set! Now you can start using Tunnel in your app. Deploy your app to a preview environment to see the Tunnel Toolbar in action.
