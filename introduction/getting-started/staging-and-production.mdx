## Add Tunnel to a hosted environment

You can use Tunnel in hosted environments using one of Tunnel's out-out-the-box components or by rendering a simple script tag. Below are instructions for React, Next.js, and other frameworks.

<Tabs>
  <Tab title="React">
    ### Install the Tunnel React SDK
    To add Tunnel to your React app, install the Tunnel React SDK using your favorite package manager.
    <CodeGroup>
      ```bash npm
      npm install @tunnel/react
      ```

      ```bash yarn
      yarn add @tunnel/react
      ```

      ```bash pnpm
      pnpm add @tunnel/react
      ```
    </CodeGroup>

    ### Find your Project ID

    <Snippet file="project-id.mdx" />

    ### Add the Tunnel Toolbar to your app
    Once you have the SDK installed, you can use the `TunnelToolbar` component to add Tunnel's collaboration features to your app.

    Below is an example of how you might add the Tunnel toolbar to your app's staging environment.
    ```jsx components/StagingToolbar.jsx
    import { TunnelToolbar } from '@tunnel/react';

    export function StagingToolbar() {
      return process.env.APP_ENV === 'staging' ? <TunnelToolbar projectId="PROJECT_ID" /> : null;
    }
    ```
    ```jsx App.jsx
    import { StagingToolbar } from './components/StagingToolbar';

    function App() {
      return (
        <>
          <h1>My App</h1>
          <StagingToolbar />
        </>
      );
    }

    export default App;
    ```

    ```jsx app/layout.jsx

      children

      return (

          <body>
            {children}

          </body>

      );
    }
    ```

      ```html

        src="https://tunnelapp.dev/__tunnel/script.js"
        data-project-id="PROJECT_ID"
      ></script>
      ```
