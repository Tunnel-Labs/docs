---
title: "Contributing"
description: "Want to help make the Tunnel Docs better?"
---

Found a bug? Grammar mistake? No problem. Let us know in our [Discord](https://discord.gg/zMw6ZF2qCf) or leave a pull request yourself.

To contribute to this documentation, simply run:

```sh
// Clone the repository
git clone https://github.com/Tunnel-Labs/docs.git

// Go into the docs folder
cd docs

// Install Mintlify
npm i -g mintlify

// Run the dev server
mintlify dev
```

Or, visit our GitHub repository: [https://github.com/Tunnel-Labs/docs](https://github.com/Tunnel-Labs/docs)
