## NPM

<Tabs>
  <Tab title="Local installation">
    <CodeGroup>

    ```bash npm
    npm install @tunnel/cli
    ```

    ```bash yarn
    yarn add @tunnel/cli
    ```

    ```bash pnpm
    pnpm add @tunnel/cli
    ```

    </CodeGroup>

  </Tab>
  <Tab title="Global installation">
    <CodeGroup>

    ```bash npm
    npm install -g @tunnel/cli
    ```

    ```bash yarn
    yarn global add @tunnel/cli
    ```

    ```bash pnpm
    pnpm add -g @tunnel/cli
    ```

    </CodeGroup>

  </Tab>
</Tabs>

## Homebrew

```bash
brew install tunnel-labs/tunnel/tunnel-cli
```
