---
title: 'Installation'
description: 'To install the Tunnel CLI, run the following command in your terminal:'
---

<Info>Requires Node.js (16.0.0 or later)</Info>

<CodeGroup>

```bash npm
npm install -g @tunnel/cli
```

```bash yarn
yarn global add @tunnel/cli
```

```bash pnpm
pnpm add --global @tunnel/cli
```

</CodeGroup>
